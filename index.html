    Frequently Asked Questions

    How do I export my TickTick backup?

    Go to TickTick Web > Avatar > Settings > Generate Backup. The backup feature is only available from the web version.

    Is my TickTick data safe?

    Yes. TickTick Insights runs entirely in your browser. No data is uploaded, stored, or tracked. The source code is fully open.

    What insights does it provide?

    Completion rates, priority distribution, tag analytics, feature usage scores, monthly activity timelines, per-folder/list breakdowns, and AI-powered personalized productivity recommendations.

    How does the AI prompt work?

    Your thousands of lines of CSV backup are condensed into a 5-15 KB structured report that AI can understand. This report requests behavioral analysis, tag assessment, list reorganization, priority usage review, and a concrete action plan.

    Which AI tools can I use with the prompt?

    You can use it with Claude, ChatGPT, Gemini, or any large language model. Just copy the prompt and paste it directly.

    Does it work with any TickTick backup?

    Yes. It supports TickTick's standard CSV backup format. File size can be up to 50 MB and UTF-8 BOM characters are handled automatically.

    Turn Your TickTick Backup Into Actionable Insights

    TickTick Insights is a free, open-source tool that transforms your TickTick backup CSV file into a detailed productivity analysis. It presents metrics like completion rates, priority distribution, tag usage patterns, feature usage scores, and monthly activity trends on a visual dashboard. From procrastination tendencies to burnout periods, neglected lists to underused features — discover the story your data tells.

    The AI Productivity Coach feature converts thousands of tasks into a structured prompt you can paste into Claude, ChatGPT, or Gemini. The AI performs behavioral analysis, suggests reorganizing your tags and lists, and creates a concrete action plan. Everything runs in your browser — your data is never sent to any server.

    Why I Built This

    My inbox was constantly overflowing. Dozens of tags, countless lists... At some point, TickTick stopped making me faster and turned into a mess I was afraid to look at. I kept thinking "if only I could organize this with AI." While writing great code with Claude Code, it hit me: why aren't we tackling the most fundamental thing — task organization with AI? I didn't want to switch to a new app — TickTick's pomodoro, mobile app, simplicity, and smart features are irreplaceable. So a simple idea was born: convert your backup into a prompt, paste it into AI, get personalized recommendations. I hope it helps you too.
